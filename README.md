# MOKAMBO PARKING AUTO ONLINE 🟢

Site web de vente de véhicules d'occasion au Congo.

## 🚀 Déploiement sur GitHub Pages

1. Créez un compte sur [GitHub](https://github.com)
2. Créez un nouveau repository nommé `mokambo-parking`
3. Uploadez tous les fichiers
4. Allez dans Settings > Pages
5. Sélectionnez la branche `main` et dossier `/ (root)`
6. Cliquez sur Save
7. Votre site sera disponible sur `https://[username].github.io/mokambo-parking/`
